# Verified Badges Directory

Place your badge images (PNG, SVG, or JPEG format) in this directory. 

To display them in the portfolio:
1. Save your badge image in this folder (e.g., `public/badges/my-badge.png`).
2. Add the badge configuration to the `badgesList` array in [CodingStats.tsx](file:///c:/Users/Aryan/Portfolio/src/sections/CodingStats.tsx):
   ```typescript
   {
     name: "My Badge Name",
     image: "/badges/my-badge.png"
   }
   ```
